# haikufy
Find haikus in text

## Install 
`npm install haikufy`

## Usage

```
Haikufy = require('haikufy');

var haikufy = new Haikufy();
haikufy.find('An old silent pond... A frog jumps into the pond, splash! Silence again.");

// Returns: 
// [ 'An old silent pond...',
//   'A frog jumps into the pond,'
//   'splash! Silence again.' ]
```
